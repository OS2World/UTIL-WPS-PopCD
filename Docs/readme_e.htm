PopCD!
    What is PopCD!? PopCD! is `Pop Up CD-ROM/Removable media/Audio CD Object' Utility. When you insert CD-ROM/Removable media/Audio CD, PopCD! creates its object on desktop and open/play. And PopCD! ejects CD-ROM/Removable Media/Audio CD when you drag and drop object into shredder.

    This OS/2 application requires Watcom VX-REXX runtime module (VROBJ.DLL) and EMX runtime modules. You can get a copy from...

    • Download VROBJ.DLL 2.1d (vrobj21d.zip)
    • Download EMX Runtime Library (0.9c or higher)
      1. Runtime Package (0.9c)
      2. Runtime Package (0.9d)

Install
  1. Unzip the archived file into any unique local directory.

  2. Run INSTALL.CMD by either OS/2 full screen or OS/2 session.
    The PopCD! program object will automatically be created in Startup Folder.
Usage
  • Basic setup

    • PopCD! detect CD-ROM media automatically, so you only do following items for basic usage.

      1. Check `Enable' check box to enable PopCD!
      2. Set delay time

    • If you want to pop up other removable media...

      1. Check `Other Removable Media' check button.
      2. Set removable drive letter immediately at removable drives entry field.
        For example, you uses removable media as drive J:, K: and L:, then set `JKL' in other removable drives entry field.

  • Icon Position

    • Check `Auto Position' radio button, if you want to manage icon position on the desktop. You can set icons inside the area you set with following parameters.

      1. Start Point X
      2. Start Point Y
      3. Rows
      4. Intervals X
      5. Intervals Y

    • Minus(-) is also valid.

  • Other parameters

    1. Volume label as object title

      • Check `Use Volume Label' check box if you want to use volume label as object title.

    2. Open folder when mount object

      • Check `Open Folder' check box if you want to open folder when mount object.

    3. Open folder view mode

      • Check `Folder View Mode' check box if you want to use `Folder View Mode' Function.
      • You can select folder open view mode from ICON/TREE/DETAILS view, and you can set some parameters of each view.

    4. CD-ROM Last Drive

      • You can set last CD-ROM drive.

    5. Ignore CD-ROM Drive

      • You can set CD-ROM drive name that you don't want to search.
        For example, you want to ignore drive P:, Q: and R:, then set `PQR' in ignore drives entry field.

    6. Caption

      • You can set defalt drive name pattern in this area.
      • `%d' will set the drive letter.
        For example, If you are a English OS/2 Warp user then set as `Drive %d' (default).

  • Audio CD

    • Basic setup

      1. Set CD Player Device ID (1-6).
      2. Check `Audio CD' check box.

    • PopCD! CD Player

      • PopCD! has internal tiny CD player. PopCD! CD Player is enabled automatically when you set Audio CD, and disabled when eject Audio CD.

      • Functions

        1. Visible CD Player
          Check `Visible CD Player' check box, if you want to pop up PopCD! CD Player.

        2. Auto Play
          Check `Auto Play' check box, if you want to play Audio CD automatically.

        3. Always on Top
          Check `Always on Top' check box, if you want to stay PopCD! CD player at top of the desktop.

        4. Key Commands
          PopCD! CD player supports following short-cut key commands.

            `P' or Right key:Play
            `S' or Left key:Stop
            `N' or Up key:Play Next Track
            `R' or Down key:Play Previous Track
            `E':Eject
            `T':Hide/Show Title Bar
            `V':Hide/Show Volume

    • External CD Player

      • You can use not only internal CD Player, also external CD Player.

  • How to Eject CD-ROM/Removable media/Audio CD media?

      You can eject CD-ROM/Removable media/Audio CD media by following ways.
      PopCD! supports Macintosh like drag & drop eject function.

      1. Drag & drop media object into shredder.
      2. Delete media object.
      3. Push drive's eject button (old way? ;->).
