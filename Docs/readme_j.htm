PopCD!
    PopCD!は自動的にCD-ROMを感知してCD-ROMのObjectをDesktopに作成するUtilityです。 自動的にCD-ROMのFolderをOpenする事も出来ます。 また、CD-ROM ObjectをシュレッダーにDrag & DropするとCD-ROMをEjectすることができます。 CD-ROM以外にもAudio CDや他のRemovable Mediaでも利用することができます。

    PopCD!を動作をさせるには別途VROBJ.DLLとEMX Runtime Libraryが必要となります。 VROBJ.DLLおよびEMX Runtime LibraryはNIFTY-Serve FOS2ADVフォーラムのLIB 9やOS/2関連の雑誌に添付のCD-ROMから入手することが出来ます。

    • Download VROBJ.DLL 2.1d (vrobj21d.zip)
    • Download EMX Runtime Library (0.9c or higher)
      1. Runtime Package (0.9c)
      2. Runtime Package (0.9d)

インストール
  1. Pcd???r.zipを適当なDirectoryにCopyし、解凍します。

  2. install.cmdを実行するとPopCD!のObjectが始動フォルダに作成されます。

使用方法
  • 簡単な使い方

    • PopCD!はCD-ROMを自動的に認識しますので、使い方は非常に簡単です。

      1. `Enable' check boxをcheckし、PopCD!をenabledにします。
      2. Delay timeを設定します。

    • 他のRemovable Mediaでの利用方法

      1. `Other Removable Media' check buttonをcheckします。
      2. Removable Driveをremovable drives entry fieldに入力します。
        J:, K:, L:をremovable mediaで使用している場合は、`JKL'を入力します。

  • Icon Position

    • Desktopの任意の領域にIconを表示する場合、`Auto Position' radio buttonを選択します。 領域は以下のParameterを使用して設定します。

      1. Start Point X
      2. Start Point Y
      3. Rows
      4. Intervals X
      5. Intervals Y

    • Minus(-)も使用することが出来ます。

  • その他の設定

    1. Volume label as object title

      • Volume LabelをObjectのTitleとして使用する場合、`Use Volume Label' check boxをcheckします。

    2. Open folder when mount object

      • Mount時にFolderをOpenする場合、`Open Folder' check boxをcheckします。

    3. Open folder view mode

      • FolderのView(ICON/TREE/DETAILS)を設定することができます。 また、各Viewの設定も可能です。

    4. CD-ROM Last Drive

      • 検索するCD-ROM Driveの範囲を変更することができます。

    5. Ignore CD-ROM Drive

      • 検索しないCD-ROMを任意に指定することができます。
        P:, Q:, R:を検索しない場合は、`PQR'を入力します。

    6. Caption

      • DefaultのDrive Nameの設定をします。
      • `%d' でDrive Letterを指定します。 例えば、日本語OS/2 Warpの場合は`ドライブ %d'と設定します(初期設定は`Drive %d'となっています)。

  • Audio CD

    • Basic setup

      1. CD PlayerのDevice ID(1-6)を設定します。
      2. `Audio CD' check boxをcheckします。

    • PopCD! CD Player

      • PopCD!はTiny CD playerを内蔵しています。 PopCD! CD PlayerはAudio CDがLoadされると自動的に起動されます。 また、Audio CDがEjectされるとCloseします。

      • Functions

        1. Visible CD Player
          PopCD! CD Playerをデスクトップに表示する場合は、`Visible CD Player' check boxをcheckします。

        2. Auto Play
          Audio CD Load時、自動的に再生する場合は、`Auto Play' check boxをcheckします。

        3. Always on Top
          PopCD! CD Playerを常時Desktopの一番上に表示させる場合は、`Always on Top' check boxをcheckします。

        4. Key Commands
          PopCD! CD playerで下記のShort-Cut Keyを使用することができます。

            `P' または Right key:再生
            `S' または Left key:停止
            `N' または Up key:次のTrackを再生
            `R' または Down key:前のTrackを再生
            `E':Audio CDをEject
            `T':Title Barの表示/非表示
            `V':Volumeの表示/非表示

    • External CD Player

      • 外部のCD Playerを利用することも可能です。

  • CD-ROMの取り出し方

      以下の方法でCD-ROMを取り出すことができます。

      1. CD-ROM objectをシュレッダーにDrag & Dropする。
      2. CD-ROM objectを削除する。
      3. CD-ROM driveのEject Buttonを押す。
